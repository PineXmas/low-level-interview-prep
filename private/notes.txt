Acutest.js is modified to fix:

- parse test result more flexibly: allowing spaces in test name, tolerate printing to console message in the result output
- allow multiple test groups syntax via: testGroups, testGroupsExeFile, testGroupsSrcFile
    Example of settings.json
    {
    "acutestExplorer.testGroups": [
        "problems/self_check",
        "problems/self_check_2"
    ],
    "acutestExplorer.testGroupsExeFile": "test.out",
    "acutestExplorer.testGroupsSrcFile": "test_runner.c",
    "acutestExplorer.logpanel": true
    }
